# 漫画元数据ComicInfo.xml的导入、编辑、生成。

**文件在html文件夹中下载。**

#### 介绍
这是一个用AI生成的轻量级 ComicInfo.xml 编辑器。
• 导入功能：上方可直接导入旧文件进行修改。
• 安全隐私：纯本地运行，无数据上传。
• 兼容性：完美支持 Komga、Tachiyomi 等。

#### 字段说明
- 系列名 (Series): 整个漫画系列的名称
- 标题 (Title): 本卷/话的具体标题
- 副标题/原名/别名 (AlternateSeries): 篇章名、日文原名或中文别名
- 话数 (Number): 第几话，如"1050"
- 卷号 (Volume): 单行本卷号，如"01"
- 语言 (Language): ISO语言代码，如zh、ja、en
- 翻译组 (ScanInformation): 记录汉化组或扫描来源
- 支持导入现有ComicInfo.xml文件自动填充

#### 字段说明:
使用方法，放到zip或cba漫画压缩文件内部的根目录下，支持的应用如Komga既可以检索到。
